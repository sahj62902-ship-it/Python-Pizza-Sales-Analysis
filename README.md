# Python-Pizza-Sales-Analysis
Short Description

A data analysis project using Python in Jupyter Notebook to analyze pizza sales data, identify trends, and generate insights to improve business performance.

Purpose

The purpose of this project is to understand sales patterns, customer demand, and revenue drivers in a pizza business so that management can make data-driven decisions related to pricing, inventory, and promotions.

Example:
Finding which pizza category generates the highest revenue helps the business focus marketing and stock planning on profitable items.

Tech Stack

Python – Data analysis and calculations

Jupyter Notebook – Interactive analysis environment

Pandas – Data cleaning and manipulation

Matplotlib / Seaborn – Data visualization

NumPy – Numerical operations

Data Source

A structured pizza sales dataset containing:

Order ID

Pizza name and category

Quantity sold

Price

Date and time of order

Example:
Each row represents a pizza sold in a particular order with quantity and price information.

Features and Highlights

Revenue calculation using quantity × price

Top-selling and least-selling pizzas

Sales by category and size

Daily and monthly sales trends

Visualizations for revenue and demand

Example:
The project identifies which pizza size (small, medium, large) sells the most and which category drives maximum revenue.

Business Impact

This analysis helps a pizza business:

Increase revenue by focusing on high-selling pizzas

Reduce waste by identifying low-demand items

Plan inventory based on demand trends

Improve promotions by targeting popular categories

Example:
If Veg pizzas generate more revenue than Non-Veg, the restaurant can run Veg combo offers to maximize sales.
